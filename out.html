Projects

  • Kilobot Simulator

    I implemeted an online Kilobot simulator. The purpose of this project is to make Kilobot programming easily and instantly accessible to everyone on all device with a modern browser. To my knowledge, this it the only Kilobot platform for running experiments in the browser and the only reproduction of the self-assembly algorithm.
  • Evolution of Morphology and Controllers for Physically Simulated Creatures

    I implemented Evolving Virtual Creatures (Karl Sims) as my undergraduate Final Year project. This project was selected by the Britich Computer Society among all final year projects to be presented in an event organized by the society.
  • No Fun Football

    An online multiplayer turn-based game that is composed of several stateless microservices, making it highly available. It is designed with scalability in mind.
  • World Cup and Derby Lotteries

    When I was at Sariina, I worked on the lottery program of a live television show during the World Cup and the most watched derby football matches. The television program, Ninety (90), was the most popular national television show in the country.
  • Creative programming

    I taught creative programming using Processing to students at the global summer school run by Institute for Advanced Architecture of Catalonia and Tehran Platform
  • No Fun Lua Engine

    I designed and developed an engine that allows running the same code on both server and client (for online games). The underlying engine is written in C++ and the game logic is written in Lua.
  • Go consultancy

    I teach programming and building services with the Go programming language. I have experience of running workshops at several institutions (companies and universities). I am available for courses and workshops.
  • Gradboostreg

    A gradient boosting regressor in Go (statistical learning)
  • Blgo

    A static file generator blog engine I wrote for my weblog.

  • Coor

    A concurrency-agnostic process coordinator and communication channel inspired by Hoare's Communicating Sequential Processes.
  • Distributed GA

    A genetic algorithm distributed runner
  • Identity and Access Management

    A flexible, performant rule-based permission mangement service for resources and users and employees at my previous job.
  • Messaging Service

    A highly concurrent system for dispatching thousands of SMS messages per second. The system allocated the limited external bandwidth among customers fairly in accordance with their subscription. I also developed the web front-end panel for this application. Languages used: Go, JavaScript, Ruby.

  • Goping

    An implementation of ICMP pinging in Go
